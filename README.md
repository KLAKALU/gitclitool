# gitclitool
clitool to setting github connection

# discrption
ssh-keyの生成からgithubへの登録、及びgithubへの疎通確認ができるCLIツールです。

# How-to-use
Windowsの場合はbatファイルをクリック、MacOSの場合はexeファイルを右クリックから開き、起動してください。  
もしくはコマンドプロンプトやターミナルなどから直接exeファイルを起動してください。
